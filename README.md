# TEKNOFEST 2026 HYZ Task 1

This repository contains the dataset, training pipeline, validation utilities and GUI-based inference system developed for TEKNOFEST 2026 HYZ Task 1.

## Features

* YOLOv8 training pipeline
* Automatic CUDA/CPU detection
* Adaptive batch size selection
* Validation and performance reporting
* Dataset integrity verification
* Image inference GUI
* Video analysis GUI
* Real-time object detection preview
* Custom model loading support

## Tech Stack

* Python
* Ultralytics YOLO
* PyTorch
* OpenCV
* NiceGUI
